# PickPath Consulting Site

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Sign in to Vercel.
4. Import the GitHub repository.
5. Deploy.

## Before going live

Edit these in `app/page.tsx`:
- `yourname@email.com`
- your About text
- any service wording you want to refine
